Curso: Autodesk Certified Profesional 3DMax

Vacantes LIMITADAS

INTRODUCCIÓN

En este curso vamos a crear scenarios y personajes digitales en 3D, realizaremos el modelado orgánico/inorgánico, texturas, rigging, animación, iluminacion, renderizado y post-producción, al terminar el curso tendrán la posibilidad de certificarse como profesional en Autodesk 3dmax 2015.

TEMAS

Módulo I
  • Introducción a la Plataforma de 3ds Max 2015
  • Interface del software / UI
  • Herramientas de navegación
  • Main toolbar, comand panel, viewport.
  • Geometrías primitivas
  • Geometrías extendidas.
  • Importación de archivos CAD.
  • Modificadores (extrude, bevel, lathe, etc)
  • Operaciones booleanas.
  • Objetos compuestos.
  • Ejercicios de aplicación.
Módulo II
  • Polygonos editables
  • Técnicas de modelado
  • Graphic Modeling tools.
  • Modelado de objetos
  • Editor de materiales
  • Parámetros básicos de un Shader.
  • Coordenadas de textura (uvmap)
  • Texturizado de objetos.
  • Animación de biped.
  • Populate (librerías de animación)
  • Animación de objetos.
  • Editor de curvas / DopSheet
  • Path Constraint.
Módulo III
  • Cámaras y parámetros principales.
  • Fundamentos de iluminación.
  • Luces fotométricas / Standard
  • Iluminación Global.
  • Iluminación interior
  • Motores de Renderizado.
  • Mental ray, Vray, iRay.
  • Render element.
  • Post-producción.
  • Preparación para certificación
  • Simulacro de examen de certificación.
